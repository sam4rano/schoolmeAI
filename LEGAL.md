## Data Usage Policy

### Data Sources

We collect data from:

1. **Public Sources**: 
   - National Universities Commission (NUC) lists
   - Institutional websites and public announcements
   - Education portals (Myschool, MySchoolGist)
   - News archives and public records

2. **User Contributions**:
   - User-reported data corrections
   - User profiles and preferences (with consent)
   - Watchlists and application tracking

3. **Scraped Data**:
   - Web scraping from public sources (respecting robots.txt)
   - Data extraction from announcements and news articles

### Data Quality

Every data point includes quality metadata:

- **Confidence Levels**: 
  - `verified`: Official source, human-verified
  - `estimated`: Calculated from multiple sources with high confidence
  - `unverified`: Scraped data, not yet verified

- **Source Provenance**: 
  - Source URL, fetch date, transformation notes
  - Version history with audit trail

- **Data Freshness**: 
  - Last verified timestamp
  - Data quality score (0-100)

### User Data Privacy

We collect and store minimal user data:

- **Required**: Email address, hashed password (for authentication)

- **Usage**: To provide personalized recommendations, watchlists, and notifications

**We do not**:
